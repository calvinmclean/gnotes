# Gnotes

This is a simple program to populate templates for weekly notes in a file structure like this:
```
2023/
  03Mar/
    Week_of_the_06th.md
```

and file contents like this:
```
# Week of the 06th (March 2023)

### Monday 06


### Tuesday 07


### Wednesday 08


### Thursday 09


### Friday 10
```
